NAMA GIM  : Dodge Game
DIBUAT    : Della — XI RPL 1 — SMK BRANTAS KARANGKATES
TAHUN     : 2026
============================================================
CARA MAIN : Buka file index.html di browser Chrome/Firefox
            Tidak perlu install apapun!
 
KONTROL   : Arrow Keys / WASD — bergerak
            ENTER — memulai gim di awal layar (splash screen)
            R — restart saat game over
            P — pause/lanjutkan jalannya gim
 
DESKRIPSI :
  Dodge Game adalah gim arkade di mana kita harus menghindar 
  dari kotak musuh yang jatuh dari atas. Bertahanlah selama 
  mungkin dari kejaran musuh yang semakin cepat, raih tingkat 
  level tertinggi, dan cetak skor terbaikmu!
 
TEKNOLOGI : HTML5 Canvas + JavaScript
            Tidak ada library eksternal
